# ExpireApp  
Aplikacja mobilna do śledzenia terminów ważności produktów spożywczych.
(Mobile application for applying expiration dates to food products.)

## Zrzuty ekranu
![Ekran główny](./screenshots/home.png)

## Funkcje
✔️ Dodawanie produktów do listy  
✔️ Organizowanie produktów według kategorii  

## Technologie
- Kotlin
- XML
- Room Database  
- Firebase Authentication

  ## Planowane funkcje
- [ ] Integracja z API do skanowania kodów kreskowych  


## Instalacja
1. Otwórz Android Studio
2. **Sklonuj repozytorium**  
3. Uruchom aplikację na emulatorze lub fizycznym urządzeniu
